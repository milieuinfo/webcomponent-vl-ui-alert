# <vl-alert> Web Component
De["alert" component van Webuniversum Vlaanderen](https://overheid.vlaanderen.be/webuniversum/v3/documentation/components/vl-ui-alert)als Web Component


## Installation
```
npm install --save vl-ui-alert
```

## Demo
```
npm run demo
```
